# Image-Viewer v1.2.3
A web image viewer jQuery plugin for desktops and phones.

## Getting Started

 * Install Image Viewer by bower

```shell
bower install image-viewer
```

 * Use it in your project

 ```javascript
$("#").imageViewer(imgURL);
```
